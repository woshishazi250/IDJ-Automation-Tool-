IDJ-Automation-Tool：可视化低代码自动化工具
IDJ 是由个人开发者开发的可视化低代码点击编程工具，核心定位是 “无需编程基础，拖拽积木即可实现电脑自动化操作”—— 通过图形化积木拼搭逻辑，让普通用户快速解放重复劳动，同时兼顾实用性与灵活性，适配多场景自动化需求。
核心功能
无需手写代码，像搭乐高一样组合 “点击、循环、等待、并行执行” 等积木块，即可让电脑自动完成：
鼠标 / 键盘模拟操作（精准定位、单击 / 双击 / 右键、毫秒级间隔调节）；
重复任务自动化（表单填写、文件批量处理、网页刷新等）；
多任务并行执行（同时运行多个自动化流程，提升效率）；
配套工具（屏幕位置选择器、截图标注、脚本调试、执行日志查看）。
适用场景
办公自动化：批量填报表、改文件名、导出数据，告别机械重复劳动；
软件测试：模拟用户点击操作，快速完成功能测试、重复场景验证；
日常辅助：网课自动播放、游戏挂机采集（请遵守对应平台规则）；
新手入门：通过积木拼搭理解编程逻辑，低门槛接触自动化技术。
